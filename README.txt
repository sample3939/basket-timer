# バスケタイマー PWA

## 概要
iPhone時計アプリ風デザインのバスケットボール試合用タイマーです。
完全オフライン対応で、音声アナウンス・ブザー音付きです。

## 機能
・5秒刻みでタイマー設定（ホイールピッカー）
・プリセット時間（7分、10分、1分、2分）
・音声アナウンス（残り◯分、30秒、10-1秒カウントダウン）
・バスケットボールブザー音（0秒時）
・一時停止・再開機能
・レスポンシブデザイン（スマホ・タブレット対応）
・完全オフライン動作（PWA）
・画面ロック防止機能

## 使用方法

### 1. ローカルサーバーで起動
```
python3 -m http.server 8000
```
ブラウザで http://localhost:8000 にアクセス

### 2. PWAとしてインストール
・ブラウザでアクセス後、「ホーム画面に追加」をタップ
・ネイティブアプリのように使用可能

### 3. 基本操作
・ホイールピッカーで時間設定（分・秒）
・プリセットボタンで瞬時設定
・Startでタイマー開始
・❚❚で一時停止、▶で再開
・Cancelで設定画面に戻る

### 4. 音声機能
・タイマー開始時に音声が自動有効化
・残り時間を日本語で音声アナウンス
・0秒でバスケットボールブザー音

## 対応ブラウザ
・Safari (iOS/macOS)
・Chrome (Android/Windows/macOS)
・Firefox
・Edge

## ファイル構成
・index.html - メインページ
・style.css - スタイル（iPhone風デザイン）
・main.js - タイマー・音声・PWA機能
・manifest.json - PWA設定
・sw.js - Service Worker（オフライン対応）
・assets/basketball_buzzer2.mp3 - ブザー音
